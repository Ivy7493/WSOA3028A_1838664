Creating a progress bar

For the threat level section of the website. I wanted to create a dynamically filling progress bar that could present the perceived threat of the character to the government. At first, I had no clue on how to do this. Some online resources suggested that I use a div as a container that you could shape into a progress bar.

For this, I created two nested section containers. The outer section container would act as the background of the bar and bar scaling factor. Now in CSS file, for the outer section, we set the background colour to white so it wouldn’t blend into the background of the website. I also set the width of the container to be 50%, so the progress bar would be centred on the screen. For the inner loop, we set the font colour to black and gave the bar an appropriate height value.

For the functionality required we would need to use some js. To determine how filled I wanted to have the bar I used a random number range from 1 to 100. Since realistically someone's threat level would fluctuate over time, using a random number meant every time someone refreshed the page, that the bar would fill to a random level. So as to give off the illusion of dynamic threat levels. The inner progress bar’s width was then scaled to the random number while keeping the outer container constant. This gave the impression of a progress bar.

Now that the progress bar was functional I decided to add some polish. Firstly I looked up how to do time intervals so I could run code over and over to give a smooth glide effect to the bar. I found the command TimeInterval, this took in a function and a period. Every period the TimeInterval variable would trigger the give function. By doing this I could increment a variable value up to the randomly selected value in which the timeInterval would then terminate. To further add to the polish of the bar. I used a switch statement to colour “lerp” the bar. Since there was no built-in interpolation function to my knowledge. So I would switch the colour every certain threshold to give the bar a bit more visual meaning for the threat level.
